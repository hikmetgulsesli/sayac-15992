# Sayac-15992 — Urun Gereksinimleri Dokumani (PRD) ## 1. Genel Bakis Sayac-15992, kullanicilarin sayisal degerleri artirma, azaltma ve sifirlama islemlerini gerceklestirebilecekleri interaktif bir web sayaci uygulamasi. ## 2. Hedefler - Saya Artirma/Azaltma/Sifirlama - Degisiklik Gecmisi (Son 10) - Tema Degisimi (Acik/Karanlik) - Veri Kaliciligi (localStorage) - WCAG 2.1 AA Erisilebilirlik - Mobil Uyumluluk ## 3. Tech Stack React, Vite, TypeScript, Tailwind CSS, Lucide Icons, Inter & Space Grotesk fonts. ## 4. Tasarim Detaylari (Playful Aesthetic) - **Sayac:** 96px Space Grotesk Bold - **Renkler:** Indigo/Violet tonlari (Primary), Slate/Zinc (Neutral) - **Radius:** 8px, 12px, 16px - **Interaksiyon:** Hover scale(1.05), Active scale(0.98)
